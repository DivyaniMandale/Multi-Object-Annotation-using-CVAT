# Indoor Multi-Object Detection using CVAT

## Project Overview

This project demonstrates image annotation for computer vision using CVAT. A custom indoor dataset containing 25 images was created and annotated using both Bounding Box and Polygon annotations. The dataset includes multiple everyday indoor objects and was exported in both YOLO and COCO formats.

---

## Tool Used

- CVAT

---

## Annotation Types

- Bounding Box
- Polygon (Instance Segmentation)

---

## Dataset Details

- Total Images: 25
- Environment: Indoor
- Annotation Tool: CVAT

---

## Export Formats

- YOLO
- COCO

---

## Project Workflow

1. Collected 25 indoor images.
2. Created labels in CVAT.
3. Annotated all objects using Bounding Box and Polygon tools.
4. Reviewed annotations for quality and consistency.
5. Exported the dataset in YOLO and COCO formats.
