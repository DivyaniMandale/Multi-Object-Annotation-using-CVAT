# Multi-Object Annotation using CVAT

## Project Overview
This project demonstrates image annotation for object detection using CVAT. A custom indoor dataset was created and annotated using bounding boxes.

## Tool Used
- CVAT

## Annotation Type
- Bounding Box

## Dataset
- 25 Indoor Images

## Object Classes
- Human
- Laptop
- Monitor
- Keyboard
- Mouse
- Mobile
- Bottle
- Coffee Mug
- Charger
- Table

## Export Formats
- YOLO
- COCO

## Project Workflow
1. Collected 25 indoor images.
2. Created labels in CVAT.
3. Annotated all objects using bounding boxes.
4. Performed quality checks.
5. Exported the dataset in YOLO and COCO formats.

## Skills Demonstrated
- Data Annotation
- Object Detection
- Bounding Box Annotation
- CVAT
- YOLO Format
- COCO Format
- Dataset Preparation
